parse (content)
Parse .env file content into a table

Parameters:

  • content string|nil The raw content of the .env file.

Returns:

  1. table|nil A table of key-value pairs.
  2. table|nil an Array of parsing warnings/errors
load (filepath)
Load and parse a .env file

Parameters:

  • filepath string|nil Path to the .env file.

Returns:

  1. table|nil Parsed environment variables or a nil if an error occurred.
  2. string|table Error message or or warnings array
export (parsed_env, options)
Export environment variables (using the most practical approach) This function generates shell scripts and instructions since lua cannot directly modify the parent shell's environment variables

Parameters:

  • parsed_env table|any The parsed environment variables
  • options table|nil Options for enxport behavior

Returns:

  1. number Number of successfully exported/processed variables
  2. table Array of error messages
  3. table Additional outputs (scripts, instructions, etc.)

Usage:

    local count, errors, outputs = lenv..export(parsed, {script_file = "set_env.sh"})
print_usage (outputs, options)
Print usage instructions for exported variables

Parameters:

  • outputs table The outputs from lenv.export()
  • options table Display options
load_and_export (filepath, options)
Convenience function to load and export in one step

Parameters:

  • filepath string Path to the .env file
  • options table|nil Export options

Returns:

  1. number Number of successfully processed variables
  2. table Array of error messages
  3. table Additional outputs

Usage:

    local count, errors, outputs = lenv.load_and_export(".env", {script_file = "set_env"})
